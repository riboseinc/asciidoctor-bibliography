Citations in AsciiDoc.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "asciidoctor-bibliography"
```

And then execute:

```sh
$ bundle install
```

Or install it yourself as:

```sh
$ gem install asciidoctor-bibliography
```
